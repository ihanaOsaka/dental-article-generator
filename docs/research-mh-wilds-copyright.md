# モンスターハンターワイルズ プレイ動画撮影・公開 著作権リサーチ

調査日: 2026-03-22

## 1. カプコン動画ガイドライン（個人向け）

カプコンは公式に「カプコン動画ガイドライン（個人向け）」を公開しており、モンハンワイルズもこのガイドラインに準拠します（ワイルズ固有の追加ガイドラインは現時点で出されていません）。

- 公式ガイドライン: https://www.capcom-games.com/ja-jp/video-policy/
- モンハンワイルズ公式FAQ: https://www.monsterhunter.com/support/wilds/ja-jp/faq/detail/057

最終更新日: 2025年12月1日

---

## 2. 許可されていること（OK）

| 項目 | 詳細 |
|------|------|
| プレイ動画の投稿・配信 | YouTube、Twitch等で自由に投稿可能 |
| エンディングまでの配信 | 発売後はエンディングを含めて配信可能 |
| 収益化 | YouTube/Twitchのパートナープログラム・広告収益はOK |
| 投げ銭 | SuperChat、Bits等の任意の投げ銭もOK（動画自体は無料公開が条件） |
| 実況・解説付き動画 | Let's Play、攻略、レビュー、リアクション、スピードラン等 |
| PC キャプチャソフトでの録画 | OBS等の外部ツール使用OK（ただし付加価値が必要） |

---

## 3. 禁止されていること（NG）

| 項目 | 詳細 |
|------|------|
| 無言プレイ動画 | 字幕・コメント・編集等の付加価値なしの単なるプレイ映像はNG |
| BGM・SE等の個別抽出 | ゲーム内音楽やボイスだけを切り出して投稿するのはNG |
| ムービーのみの投稿 | ゲーム内カットシーンだけをそのまま投稿するのはNG |
| 公式映像の転載 | PV・トレーラー等の公式プロモーション映像の無断転載はNG |
| 有料限定公開 | 動画を有料でのみ視聴可能にするのはNG |
| 公序良俗に反するMOD・コンテンツ | 差別的・性的・違法な内容のMODを使用した動画はNG |
| 発売前コンテンツの公開 | リーク映像等は厳禁 |
| 公式コンテンツとの誤認 | ファンコンテンツを公式と偽るのはNG |

---

## 4. 個人 vs 法人の区別（重要）

- **個人**: ガイドライン遵守で申請不要
- **法人・事務所所属者**: カプコンの個別許諾が必要
  - ただし、所属先がすでにカプコンの許諾を得ている法人・MCN事業者の場合は個別申請不要

→ 個人として活動する場合は、ガイドラインを守れば追加の許可申請は不要です。

---

## 5. BGM・音楽に関する注意点

### ゲーム内BGMについて
- ゲームプレイに伴う録画にゲーム内BGMが含まれるのは **問題なし**
- ただし、BGMだけを抽出して「BGM集」として投稿するのは **禁止**

### 外部BGM・音楽の使用
- 配信中に市販の音楽（CD、Spotify等）を流す場合、**原盤権の侵害**になるリスクあり
- JASRAC/NexToneの包括契約は「著作権」のみで「原盤権」はカバーされない
- カプコンのゲーム音楽はJASRAC管理外のものが多い
- **推奨**: フリーBGM素材を使用するか、ゲーム内BGMのみで録画する

---

## 6. PC（Steam版）固有の注意点

- OBS等のキャプチャソフトでの録画はOK
- ただし「独自の付加価値」（実況、字幕、編集等）を付けることが求められる
- MODを使用した動画は、MODの内容が公序良俗に反しない限りはグレーゾーン
  - チート系MODの使用は注意が必要（ガイドライン上、明確な禁止ではないが推奨されない）

---

## 7. 実践チェックリスト

動画を公開する前に確認すべき項目:

- [ ] 実況音声または字幕・編集等の付加価値を付けているか
- [ ] ゲーム内ムービーだけの投稿になっていないか
- [ ] BGMを個別に抽出していないか
- [ ] 公序良俗に反するMODを使用していないか
- [ ] 動画を有料限定にしていないか（投げ銭はOK、有料限定はNG）
- [ ] 公式コンテンツと誤認されるような表示をしていないか
- [ ] ネタバレを避けたい人への配慮（タイトル・サムネイル・冒頭に注意書き推奨）
- [ ] 外部BGMを使用する場合、著作権・原盤権をクリアしているか
- [ ] 個人としての活動か（法人・事務所所属の場合は別途許諾が必要）
