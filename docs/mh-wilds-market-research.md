出典: CESAゲーム産業レポート2025

### 年齢別の動向（インテージ調査 2024年）

- 15~59歳の**54.0%**（推計3,551万人）が年1日以上ゲームをプレイ
- **20代以下**: ゲーム人口が減少傾向（スマホ・家庭用ともに減少、PCのみ増加）
- **30代**: コロナ禍で最もライフスタイル変化が大きかった世代。家庭用・PCゲームの中核ユーザー層
- **40~50代**: 家庭用ゲームの支出率が増加傾向
- **50代**: スマホゲームの「毎日プレイ率」が全年代中最高の**72%**

### PCゲームの潜在ユーザー（ゲームエイジ総研 2023年）

- PCゲームプレイ経験者: **1,052万人**
- PCゲーム潜在ユーザー（未プレイだが興味あり）: **1,126万人**
- **男性20代と40代が最多**（各150万人超）、男女比は男性69.5%:女性30.5%

## 1.3 30~50代の「潜在ゲーマー」分析

### ゲームをやらない/やめた主な理由

| 理由 | 詳細 |
|---|---|
| **時間がない** | 仕事・育児・家事で自由時間が確保できない（最大の要因） |
| **心の余裕がない** | 仕事・将来・お金の不安で精神的余裕が低下 |
| **集中力・体力の低下** | 長時間プレイがしんどい |
| **起動の億劫さ** | ゲーム機の起動すら面倒で、スマホやYouTubeに流れる |
| **きっかけがない** | 何を遊べばいいかわからない、周囲にゲーマーがいない |
| **新鮮さの低下** | グラフィックや内容に驚きを感じなくなった |

### コロナ禍以降の復帰トレンド（ゲームエイジ総研 2021年調査）

- **51.2%** のゲーマーが「プレイ時間が増えた」
- **20.3%** が「今までプレイしていなかったが、ゲームを始めた」
- **最も変化したのは30代**
- 国内ゲーム市場は2020年に初めて**2兆円を突破**（前年比17%増）

### 動画で使えるエビデンスポイント

> 「PCゲーム潜在ユーザー1,126万人のうち、40代男性が150万人超で最大ボリュームゾーンの一つ。
> 私はまさにそのど真ん中にいるわけです。」

# Part 2: YouTube ゲーム実況 競合分析・市場トレンド

## 2.1 モンハンワイルズ実況の競合状況

### 主要な競合チャンネル

| チャンネル名 | 登録者数 | スタイル |
|---|---|---|
| **よしなま** | 約103万人 | ハイテンション・ツッコミ実況。モンハン古参 |
| **茶々茶** | 約60万人 | 情報系＋実況のベテラン |
| **2BRO.（兄者弟者）** | 約315万人 | マルチプレイ・ワイワイ系 |
| **タカティン** | 中規模 | 太刀特化の攻略系 |
| **GaRaNa** | 中規模 | 片手剣特化・高スキル系 |
| **Mハシ** | 中規模 | 寸劇・ネタ系 |

### 人気スタイルの分類

1. **ガチ攻略・武器特化系**: 特定武器種に特化
2. **ハイテンション・エンタメ系**: トーク力とリアクション
3. **情報・ニュース系**: 最新情報のまとめ
4. **ネタ・寸劇系**: 企画力勝負
5. **参加型マルチ配信**: 視聴者参加

### 「初見プレイ」系の現状

- 「初見歓迎」「完全初見」タグの配信は**非常に多数**存在
- ただし大半は小規模チャンネルのライブ配信
- 編集された初見プレイ動画で高再生数を獲得しているチャンネルは**限定的**

### 視聴者が重視するもの

- **「実況者の人柄・トークが面白いから」が全体の約6割で最多**
- ゲーム内容そのものよりも実況者の魅力が重視される（ドズル社調査）

### 2024~2026年のトレンド変化

- VTuberの存在感がさらに拡大
- 付加価値型コンテンツ（考察・企画・コミュニティ連動）が伸びる傾向
- 芸能人参入の定着（狩野英孝42歳が236万人）
- ゲーム関連トピックが年間ランキング上位に多数 → ジャンル全体として拡大傾向

### VTuber vs 顔出し

- **VTuber**: キャラクター性・年齢変化に左右されないブランド力
- **顔出し**: 親近感・信頼感で差別化。ゲーム実況者の中では**少数派**のため目立てる
- 両者の境界線は曖昧化中

## 2.3 ニッチ実況チャンネルの成功パターン

### 本業 x ゲーム実況の事例

- **弁護士 x ゲーム**: 落合一樹弁護士がゲーム法務の専門知識を活かしたチャンネル運営
- **芸能人 x ゲーム**: 狩野英孝（42歳）が236万人 — 年齢をむしろ武器に
- **医師のゲーム実況の大きな成功事例は確認できず** → 参入すれば目立てる可能性大

### 「おじさんゲーマー」「社会人ゲーマー」チャンネル

- **明確にこのコンセプトで成功している大型チャンネルが見当たらない**
- → **ブルーオーシャンに近い可能性がある**

### 後発参入で伸びるチャンネルの特徴

1. **ジャンル・タイトル特化**: 1つのゲームに絞り込む
2. **ブーム後も継続**: 3~6ヶ月で離脱する多数の実況者の後に残る
3. **サムネイルの質**: 編集の質よりもクリック率
4. **ショート動画の併用**: ハイライトをショートとして切り出し
5. **初期は編集動画優先**: ストック型資産として有利

### 低頻度更新でも成功する条件

- **週1~2回でも可能**: 「質の高さ」「特定ゲーム特化」「ショート動画併用」が条件
- 教育的・ドキュメンタリー的コンテンツであれば月1~2回でも効果あり

## 2.4 ゲーム実況の流行り廃り

### 今伸びているジャンル（2025年）

| ジャンル | YouTube再生数規模 | 理由 |
|---|---|---|
| インディーゲーム | 約1,819万回 | 新鮮味、単発向き |
| ホラーゲーム | 約1,723万回 | リアクション映え |
| シミュレーション | 約518万回 | 視聴者参加型と相性良 |

### モンハン実況の寿命サイクル

| 時期 | 状況 |
|---|---|
| **発売~3ヶ月** | 最大のピーク。大量参入 |
| **3ヶ月~半年** | 大抵の実況者が離脱。同接138万→50万台 |
| **半年~1年** | アップデートで一時回復（例: FF14コラボ） |
| **1~2年後** | **大型拡張DLCで第二のピーク**（2026年夏予定） |
| **2年以上** | 新作発売で低下。ただしシリーズ発表で再ブレイクの可能性 |

### 重要な示唆

**今から参入しても、2026年夏の大型DLC発売時に「第二波」に乗れるタイミング。**
むしろ「発売から1年かけてゆっくり進めた」ストーリーが差別化要因になりうる。

# Part 3: 総合分析 — チャンネルのポジショニング

## 3.1 潜在層プロファイルまとめ

| 指標 | 数値 |
|---|---|
| PCゲーム潜在ユーザー | **1,126万人** |
| うち40代男性 | **150万人超**（最大ボリュームゾーンの一つ） |
| コロナ禍での新規ゲーマー率 | **20.3%** |
| 離脱理由の最大要因 | 時間がない・きっかけがない |
| Steam日本ユーザー | **約355万人**（急成長中） |

## 3.2 競合環境まとめ

| 領域 | 競合密度 | 備考 |
|---|---|---|
| モンハン攻略系 | **高** | 既にベテラン多数 |
| モンハン初見実況 | **中** | 多いが大型チャンネルは少ない |
| おじさんゲーマー | **低** | ブルーオーシャン |
| 専門職 x ゲーム | **極低** | ほぼ未開拓 |
| 歯科医師 x ゲーム | **ゼロ** | 唯一無二 |

1. 「PCゲーム潜在ユーザー1,126万人。40代男性が150万人超で最大ボリュームゾーン。私はど真ん中です」
