## 1.3 30~50代の「潜在ゲーマー」分析

### ゲームをやらない/やめた主な理由

| 理由 | 詳細 |
|---|---|
| **時間がない** | 仕事・育児・家事で自由時間が確保できない（最大の要因） |
| **心の余裕がない** | 仕事・将来・お金の不安で精神的余裕が低下 |
| **集中力・体力の低下** | 長時間プレイがしんどい |
| **起動の億劫さ** | ゲーム機の起動すら面倒で、スマホやYouTubeに流れる |
| **きっかけがない** | 何を遊べばいいかわからない、周囲にゲーマーがいない |
| **新鮮さの低下** | グラフィックや内容に驚きを感じなくなった |

### コロナ禍以降の復帰トレンド（ゲームエイジ総研 2021年調査）

- **51.2%** のゲーマーが「プレイ時間が増えた」
- **20.3%** が「今までプレイしていなかったが、ゲームを始めた」
- **最も変化したのは30代**
- 国内ゲーム市場は2020年に初めて**2兆円を突破**（前年比17%増）

### 動画で使えるエビデンスポイント

> 「PCゲーム潜在ユーザー1,126万人のうち、40代男性が150万人超で最大ボリュームゾーンの一つ。
> 私はまさにそのど真ん中にいるわけです。」

### 視聴者が重視するもの

- **「実況者の人柄・トークが面白いから」が全体の約6割で最多**
- ゲーム内容そのものよりも実況者の魅力が重視される（ドズル社調査）

### 2024~2026年のトレンド変化

- VTuberの存在感がさらに拡大
- 付加価値型コンテンツ（考察・企画・コミュニティ連動）が伸びる傾向

### VTuber vs 顔出し

- **VTuber**: キャラクター性・年齢変化に左右されないブランド力
- **顔出し**: 親近感・信頼感で差別化。ゲーム実況者の中では**少数派**のため目立てる
- 両者の境界線は曖昧化中

## 2.3 ニッチ実況チャンネルの成功パターン

### 本業 x ゲーム実況の事例

- **弁護士 x ゲーム**: 落合一樹弁護士がゲーム法務の専門知識を活かしたチャンネル運営
- **芸能人 x ゲーム**: 狩野英孝（42歳）が236万人 — 年齢をむしろ武器に
- **医師のゲーム実況の大きな成功事例は確認できず** → 参入すれば目立てる可能性大

### 「おじさんゲーマー」「社会人ゲーマー」チャンネル

- **明確にこのコンセプトで成功している大型チャンネルが見当たらない**
- → **ブルーオーシャンに近い可能性がある**

### 後発参入で伸びるチャンネルの特徴

1. **ジャンル・タイトル特化**: 1つのゲームに絞り込む
2. **ブーム後も継続**: 3~6ヶ月で離脱する多数の実況者の後に残る
3. **サムネイルの質**: 編集の質よりもクリック率
4. **ショート動画の併用**: ハイライトをショートとして切り出し
5. **初期は編集動画優先**: ストック型資産として有利

### 低頻度更新でも成功する条件

- **週1~2回でも可能**: 「質の高さ」「特定ゲーム特化」「ショート動画併用」が条件
- 教育的・ドキュメンタリー的コンテンツであれば月1~2回でも効果あり

## 2.4 ゲーム実況の流行り廃り

### 今伸びているジャンル（2025年）

| ジャンル | YouTube再生数規模 | 理由 |
|---|---|---|
| インディーゲーム | 約1,819万回 | 新鮮味、単発向き |
| ホラーゲーム | 約1,723万回 | リアクション映え |
| シミュレーション | 約518万回 | 視聴者参加型と相性良 |

### モンハン実況の寿命サイクル

| 時期 | 状況 |
|---|---|
| **発売~3ヶ月** | 最大のピーク。大量参入 |
| **3ヶ月~半年** | 大抵の実況者が離脱。同接138万→50万台 |
| **半年~1年** | アップデートで一時回復（例: FF14コラボ） |
| **1~2年後** | **大型拡張DLCで第二のピーク**（2026年夏予定） |
| **2年以上** | 新作発売で低下。ただしシリーズ発表で再ブレイクの可能性 |

### 重要な示唆

**今から参入しても、2026年夏の大型DLC発売時に「第二波」に乗れるタイミング。**
むしろ「発売から1年かけてゆっくり進めた」ストーリーが差別化要因になりうる。

## 2.5.2 狩野英孝「EIKO!GO!!」のスタイル詳細分析

### 基本データ

| 項目 | 内容 |
|------|------|
| 本名/芸名 | 狩野英孝 |
| 生年 | 1982年（2026年現在43歳） |
| チャンネル名 | EIKO!GO!! |
| 登録者数 | 273万人 |
| 自称 | 「プレイステーション世代」 |
| ファン層 | **7割が30代〜40代**、男女比6:4 |
| 芸能人ゲーム実況好感度 | **1位（22.8%）**、2位 有吉弘行14.6%、3位 本田翼14.3% |

### 「年齢を武器にする」の具体的内容

1. **懐かしさの共有**: SFC→PS→バイオハザードという同世代共通のゲーム原体験を持ち、「あの頃」の感覚を視聴者と共有
2. **「上手くないこと」が自然**: 若い世代と違い反射神経・操作巧みさで勝負しない。40代が真剣にやっても上手くいかない姿 → 同世代に「自分もこうなるだろう」という親近感
3. **かつてクリアできなかったゲームへの再挑戦**: 中学時代に怖くてギブアップしたバイオハザードに大人になって再挑戦する構図自体がコンテンツに

### ゲームスキル：「壊滅的に下手」が最大の武器

- 本人談: 「へたくそだからゲーム実況やりたくなかった」
- Diamond Online: 「壊滅的に下手でも魅せる芸人スキル」
- 文春オンライン: 「ゲームは下手くそなのに人気再燃」
- **真面目にプレイ→予想外のゲームオーバー連発→情緒不安定**、がエンタメに
- コントローラー故障で斧を振り回し続ける「斧事件」がTwitterトレンド入り
- 配信プロデューサー評: 「"普通のミスやハプニング"が"ミラクルに映る"」「小さなハプニングを大きな笑いに変える事ができる」

### 芸人スキルの活かし方

| ポイント | 詳細 |
|----------|------|
| ピン芸人のロケ経験 | 一人でゲーム画面に向かいながら自然にトーク展開できる |
| ドッキリで鍛えた裏表のないリアクション | 驚いてヘッドフォンをふっ飛ばすなど身体的リアクションも自然体 |
| **ユーザー名を読む** | 他の実況者がコメントだけ読む中、狩野は名前もしっかり読む → ラジオのペンネーム的な親密感 |
| 腰の低さ | 「視聴者を下に見ないで腰が超低い」と評価 |
| 上手い人のアドバイスを素直に受け入れる | 「応援したくなるキャラクター」として機能 |

### モンハンとの関わり

- PSP時代からの大ファン（飛行機キャンセルして新幹線でプレイするほど）
- **モンハンライズ実況の伝説**: ターゲットモンスターを間違えて約40分間別のモンスターと戦い続けクエスト失敗 → Twitterで「トビカガチ」がトレンド入り
- **モンハンワイルズ**: カプコン公式番組「カプコンTV!!」のメインパーソナリティ。ベータテストを7時間配信
- 発売日生配信中にPS5がクラッシュ→「No Signal」がモンスター扱いされネタに
- **狩野の実況を見て「買うことを決めた」視聴者が多数** → 販促力として注目

### 当チャンネルへの示唆（狩野英孝との差別化と参考ポイント）

| 狩野英孝 | 当チャンネル | 差別化ポイント |
|----------|-------------|----------------|
| 芸人（エンタメのプロ） | 歯科医師（専門職） | 職業ギャップの方向性が異なる |
| ゲーム下手だが経験豊富 | ゲーム自体が**完全未経験** | 初見の純度が圧倒的に高い |
| 大量の視聴者との対話 | データ・エビデンスベースのトーク | 知的好奇心層に訴求 |
| 「笑い」がコア | 「共感・発見」がコア | 視聴後の感情が異なる |
| 週複数回の高頻度配信 | 週1〜2回の質重視 | 忙しい同世代に合わせたペース |

**参考にすべき点:**
- 「下手でも楽しめる」というメッセージは共通して有効
- ユーザー名を読む・視聴者を大切にする姿勢は必須
- ハプニングを隠さず活かす姿勢
- 同世代の「あるある」を自然に組み込む
