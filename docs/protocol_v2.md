# エビデンス検索・記事生成プロトコル v2

## v1で発生した問題

### 根本原因: 介入の不一致（Intervention Mismatch）

「フッ素塗布」の記事を書く際に、「fluoride varnish（フッ化物バーニッシュ）」の
エビデンスを「日本のフッ素塗布（APFゲル塗布）」のエビデンスとして使用した。

- **Fluoride varnish**: NaF 22,600ppm、樹脂基材で歯面に付着、数時間持続
- **日本の保険フッ素塗布**: APFゲル 9,000ppm、トレー/綿球法、3-4分間塗布

これらは全く異なる介入であり、エビデンスの直接適用は不適切。

### 問題の連鎖

1. **検索段階**: "fluoride" で一括検索し、delivery method を区別しなかった
2. **スクリーニング段階**: 論文のPICOと記事のPICOの照合を行わなかった
3. **執筆段階**: エビデンスの適用可能性（applicability）を検討しなかった
4. **チェック段階**: 介入の一致を確認する仕組みがなかった

## v2 プロトコル

### Phase 1: トピック定義の精緻化

**変更点**: PICOの「I（介入）」を具体的に定義し、「日本の臨床実態」を明記する。

```yaml
- id: preschool_fluoride_application
  title: "歯医者でのフッ素塗布、本当に効果あるの？"
  japan_clinical_reality:
    insurance_intervention: "APFゲル歯面塗布（9,000ppm, フルオール・ゼリー等）"
    insurance_code: "I031 フッ化物歯面塗布処置"
    frequency: "3ヶ月に1回（か強診は月1回）"
    alternative_interventions:
      - name: "フッ化物バーニッシュ（NaF 22,600ppm）"
        insurance: false
        note: "日本では知覚過敏抑制剤として承認。う蝕予防は適応外使用"
  pico:
    population: "3〜6歳の幼児"
    intervention_primary: "APFゲル歯面塗布（日本の保険診療）"
    intervention_alternative: "フッ化物バーニッシュ（自費）"
    comparison: "フッ化物塗布なし"
    outcome: "むし歯予防効果（DMFT/dmft）"
```

### Phase 2: エビデンス検索（介入別に分離）

**ルール**: 介入ごとに別の検索戦略を立てる。混ぜない。

```
検索1: 日本の保険介入に対応するエビデンス
  → "APF gel" OR "acidulated phosphate fluoride" AND caries prevention
  → Cochrane: Marinho 2015 (CD002280) — APFゲルのレビュー

検索2: 代替介入（バーニッシュ）のエビデンス
  → "fluoride varnish" AND caries prevention
  → Cochrane: Marinho 2013 (CD002279) — バーニッシュのレビュー

検索3: 直接比較
  → "fluoride varnish" vs "fluoride gel" comparison
```

### Phase 3: エビデンス-介入 照合チェック（新設）

記事執筆前に以下のチェックリストを実行する:

```
□ 引用する各エビデンスの介入（I）は、記事で述べる介入と一致しているか？
  - 一致: そのまま使用可
  - 類似だが異なる: 「〇〇の研究では△△を使用しており、
    日本で行われる□□とは異なるが...」と明記
  - 不一致: 使用不可（または比較対象としてのみ言及）

□ 日本の臨床実態との乖離はないか？
  - 使用材料の違い
  - 適用頻度の違い
  - 保険適用の有無
  - 対象年齢の違い

□ エビデンスの質に関する注意点は明記したか？
  - I²（異質性）が50%以上 → 「研究間のばらつきが大きい」と明記
  - バイアスリスクが高い研究が多い → 「効果が過大評価されている可能性」と明記
  - 盲検化が困難 → その旨明記
  - 未更新のレビュー → 「〇年時点のレビュー」と明記
```

### Phase 4: 記事構成テンプレート（改訂）

```markdown
# [惹きつけるタイトル]

[フック — 1文目で引き込む]

## 結論（先に結論を述べる）

## 日本の歯科医院で行われる「フッ素塗布」とは
- 具体的に何をしているのかを説明
- 使用材料、方法、頻度

## エビデンス：何がどこまで分かっているか
- 日本で行われている方法（APFゲル等）のエビデンス
- 世界標準（バーニッシュ）のエビデンス
- 両者の違いを正直に説明

## 解釈上の注意点（批判的吟味セクション — 必須）
- 研究の限界を正直に述べる
- 「〇〇だから効果がない」ではなく
  「〇〇という限界はあるが、現時点では△△と考えられる」

## 当院からのご提案
- 保険のフッ素塗布 + 自費のバーニッシュなど
- 個別のリスク評価に基づくプラン

## まとめ

## 参考文献
- 介入の種類を明記: [varnish] [APF gel] [guideline] 等
```

### Phase 5: 品質チェック（改訂）

```
□ 介入一致チェック: すべてのエビデンスが正しい介入に紐づいているか
□ 批判的吟味セクション: 存在するか、具体的か
□ 日本の臨床実態: 正確に記述されているか
□ 保険/自費の区別: 明記されているか
□ 誇大表現チェック: 「最強」「完全に」「絶対」等の表現がないか
□ 医療広告GL準拠: 「最高」「最良」等の比較優良広告に該当しないか
□ 出典の透明性: すべての数値に出典があるか
```

---

## 教訓

1. **「フッ素」で一括りにしない** — delivery method が異なれば別の介入
2. **日本の臨床実態を最初に調べる** — 保険点数・材料・方法を確認してからエビデンス検索
3. **エビデンスの適用可能性を常に問う** — "Does this evidence apply to THIS intervention?"
4. **限界は隠さない** — 正直さが信頼を生む。批判的吟味セクションは必須
5. **「世界標準」と「日本の実態」のギャップ** — ギャップがある場合は、むしろ記事の独自性になる
