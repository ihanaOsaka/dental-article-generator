# シェーグレン症候群と口腔の健康管理

**対象：シェーグレン症候群を持つ成人の方とそのご家族へ**

---

## はじめに

シェーグレン症候群（Sjögren's disease; SjD）は、唾液腺・涙腺などの外分泌腺にリンパ球が浸潤する慢性の全身性自己免疫疾患です。主な症状は**口の乾燥（ドライマウス・口腔乾燥症）**と**目の乾燥（ドライアイ）**ですが、関節痛・倦怠感・末梢神経障害など、全身に多彩な症状をきたすことがあります[^15][^7]。

日本では厚生労働省の指定難病となっており、2017年に日本独自のシェーグレン症候群診療ガイドラインが策定されています[^19]。現時点では根治療法はなく、治療の中心は症状のコントロールですが、近年は免疫療法を中心に新たな治療薬の研究が急速に進んでいます[^2]。

この記事では、とくに**口腔（お口）の健康管理**に焦点を当て、患者さんやご家族が日常生活で実践できるセルフケアと、歯科・医科での最新の対応について解説します。

---

## シェーグレン症候群が口腔に及ぼす影響

### ドライマウス（口腔乾燥症）とは

シェーグレン症候群では、免疫細胞（リンパ球）が唾液腺を攻撃するため、**唾液の分泌量が著しく低下**します[^15]。唾液は単なる「水分」ではなく、以下の重要な役割を担っています。

| 唾液の役割 | 低下したときに起こること |
|---|---|
| 食物の消化補助・飲み込みやすくする | 食事が困難になる |
| 口腔内の細菌・酸を洗い流す | むし歯・歯周病のリスク上昇 |
| 抗菌作用（リゾチーム等） | 口腔カンジダ症などの感染症増加 |
| 緩衝作用（pH調整） | 歯の酸蝕・むし歯が進みやすくなる |
| 粘膜保護 | 口内炎・粘膜びらんの発生 |
| 発音・会話の補助 | 話しにくくなる |

### なぜむし歯になりやすいのか

唾液の分泌が減ることで口腔内の自浄作用が失われ、酸性環境が続くと歯の表面（エナメル質）が溶け始めます。さらに口腔内の細菌バランスが乱れるため、シェーグレン症候群の患者さんでは**むし歯（齲蝕）の発症リスクが健常者と比較して著しく高い**とされています。とくに歯頸部（歯と歯ぐきの境目）や歯の切端（前歯の先端）など、通常はむし歯になりにくい部位にも多発する「**放射線性う蝕に類似したパターン**」が見られることがあります。

## 日常でできる口腔ケア：セルフケアの実践

### 1. こまめな水分補給と保湿

口腔乾燥の不快感を和らげる最も基本的な対処法は、**水を少量ずつ頻繁に口に含む**ことです。ただし、砂糖を含む飲料はむし歯を促進するため避けてください。

- **人工唾液・口腔保湿ジェル**：薬局などで入手できる保湿製品（スプレータイプ・ジェルタイプ）を就寝前や食事前後に使用すると、乾燥感の軽減に有効です。体系的なレビューでは、これらの外用製品がドライマウスの自覚症状改善に一定の効果を示すことが報告されています[^21]。

### 2. 唾液腺を刺激する

唾液腺がまだ残存機能を持つ場合、物理的・化学的刺激によって唾液分泌を促せることがあります。

- **無糖のキシリトールガムや無糖キャンディーを噛む**：咀嚼刺激と甘み刺激が唾液分泌を促します。キシリトールはむし歯の原因菌（ミュータンス菌）の増殖を抑える効果もあり、ドライマウスの患者さんに特に適しています。
- **梅干しやレモン（少量）などの酸味**：唾液腺への刺激になりますが、歯の酸蝕を防ぐために食後はよくすすぐことが大切です。
- **唾液腺マッサージ**：耳下腺（耳の前下方）・顎下腺（顎の内側）・舌下腺（舌の下）を指で優しくマッサージすることで唾液分泌が促されることがあります。

### 3. フッ化物の積極的活用

フッ化物（フッ素）は、歯の再石灰化を促し、むし歯菌の活性を抑制する効果があります。ドライマウスの患者さんでは、**高濃度フッ化物配合歯磨き剤**（1450ppm以上）の使用や、**フッ化物ジェルのホームトレー塗布**が有効な予防策として推奨されます。歯科医師に相談し、個人に合ったフッ化物プログラムを組んでもらいましょう。

> **参考**：WHOのう蝕予防マニュアルでも、フッ化物の適切な使用が歯科疾患予防の中心的な介入であると明記されています[^1]。

### 4. 丁寧な歯磨きと口腔清潔の維持

- **歯磨きは1日2〜3回**、特に就寝前は念入りに行う。
- **電動歯ブラシ**は手磨きよりもプラーク除去効率が高く、手指の動きに不自由がある方にも有効です。
- **デンタルフロス・歯間ブラシ**：歯と歯の間のプラーク除去に欠かせません。
- **アルコール含有の洗口液は避ける**：アルコールは口腔粘膜をさらに乾燥させます。アルコールフリーの洗口液を選びましょう。
- **抗菌洗口液（クロルヘキシジン等）**：歯科医師の指示のもと、口腔カンジダ症予防のために使用する場合があります。

### 5. 食事・栄養のコントロール

2025年の系統的レビューでは、シェーグレン症候群患者に対する栄養介入（抗炎症食・オメガ3脂肪酸・抗酸化物質の摂取など）が症状改善に一定の可能性を示すことが報告されています[^5]。口腔管理の観点から特に重要な食事上の注意点は以下のとおりです。

- **糖分の多い食品・飲料を控える**：特に甘い飲み物を少しずつ長時間かけて飲む習慣は非常に危険です。
- **粘り気のある食品（餅・キャラメルなど）を避ける**：口腔粘膜への刺激と歯への付着リスクが高まります。
- **柔らかく水分を含んだ食事を工夫する**：乾燥した食品は飲み込みにくく、粘膜を傷つけることがあります。スープや煮物など汁気のあるものを積極的に取り入れましょう。

## 歯科・医科での治療：専門的な介入

### 薬物療法：唾液分泌促進薬

#### ピロカルピン（Pilocarpine）

ピロカルピンはムスカリン受容体作動薬で、唾液腺・涙腺などの外分泌腺を直接刺激することで分泌を促します。2024年に発表されたランダム化比較試験（RCT）のメタアナリシスでは、**ピロカルピンの経口投与がシェーグレン症候群の口腔乾燥症状および唾液分泌量を有意に改善する**と結論づけられました[^6]。日本のシェーグレン症候群診療ガイドライン2017でも、唾液分泌促進薬として推奨されています[^19]。

主な副作用として発汗・悪心・頻尿などがあるため、医師と相談しながら適切な用量を決める必要があります。

#### セビメリン（Cevimeline）

ピロカルピンと同様のムスカリン受容体作動薬で、日本でも保険適用されています。欧州リウマチ学会（EULAR）の推奨においても、局所的・全身的な唾液分泌促進薬として位置づけられています[^18]。

### 唾液腺内視鏡（唾液腺鏡・Sialendoscopy）

唾液腺内視鏡は、細い内視鏡（シャロエンドスコープ）を唾液腺の導管口から挿入して導管内を直接観察・治療する低侵襲な処置です。2021年の系統的レビューでは、**唾液腺内視鏡と生理食塩水・ステロイドなどによる洗浄がシェーグレン症候群患者の唾液分泌機能を改善する可能性**があることが示されました[^16]。ただし対象施設が限られており、専門的な技術を要する処置です。主要な医科大学病院や耳鼻咽喉科・口腔外科で相談してみましょう。

### 局所フッ化物塗布・シーラント（歯科でのむし歯予防処置）

歯科医院では、高濃度フッ化物ワニスの塗布やシーラント（奥歯の溝を塞ぐ処置）を定期的に行うことで、ドライマウスによるむし歯多発リスクを軽減できます。特に唾液分泌が著しく低下している場合は、**カスタムトレーによる高濃度フッ化物ジェルのホームケア**を処方してもらうことも有効です。

### 鍼灸療法（補完代替医療）

2018年の系統的レビューによると、**鍼治療がドライマウスおよび唾液分泌低下に対して一定の改善効果を示す**可能性があることが報告されています[^20]。ただし、研究の質にばらつきがあり、現時点ではエビデンスが十分ではないため、補助的な選択肢として慎重に検討するレベルにとどまります。

### 新規治療・研究の動向

2025年の最新の系統的レビューでは、生物学的製剤（JAK阻害薬・抗CD20抗体・抗BAFF抗体など）をはじめとする**新規の疾患修飾薬の開発**が急速に進んでいることが報告されており、根本的な病勢コントロールへの期待が高まっています[^2]。治療法の選択は主治医（内科・リウマチ科）と相談してください。

## 定期的な歯科受診の重要性

シェーグレン症候群患者さんでは、以下の理由から**通常よりも高頻度な歯科受診（3〜4ヶ月ごと）**が推奨されます。

1. **早期むし歯の発見と治療**：ドライマウスにより急速にむし歯が進行する場合があります。
2. **歯周病の管理**：口腔内の細菌バランスの乱れにより歯周病も起こりやすくなります。
3. **口腔カンジダ症のスクリーニング**：唾液の減少は口腔内の真菌感染リスクを高めます。
4. **口腔粘膜の観察**：シェーグレン症候群患者は悪性リンパ腫などの発症リスクがわずかに高いことが報告されており[^4]、口腔粘膜の異常な変化を早期に察知することが重要です。
5. **フッ化物塗布・専門的クリーニング**：自宅ケアの補完として定期的なプロフェッショナルケアが不可欠です。

## まとめ：口腔管理のポイント

| 場面 | 実践すること |
|---|---|
| 毎日のセルフケア | 高濃度フッ化物歯磨き剤使用、丁寧な歯磨き・フロス、保湿ジェル・人工唾液の活用、こまめな水分補給 |
| 食生活 | 糖分・酸性食品を控える、水分を含んだ食事を工夫する、キシリトールガムの活用 |
| 歯科受診 | 3〜4ヶ月ごとの定期受診、フッ化物塗布・プロフェッショナルケア |
| 医科（内科・リウマチ科）との連携 | 唾液分泌促進薬（ピロカルピン・セビメリン）の処方相談 |
| 生活習慣 | 禁煙・節酒（粘膜乾燥を悪化させる）、加湿器の活用 |

シェーグレン症候群の口腔管理は、歯科と医科が**連携してチームとして取り組む**ことが最も重要です。症状や不安なことがあれば、担当の歯科医師や主治医に遠慮なく相談してください。
