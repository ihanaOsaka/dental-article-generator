# 口呼吸が歯並びを悪くする？ — 鼻呼吸の大切さのエビデンス

**お子さんが普段口を開けていませんか？口呼吸は歯並びだけでなく、顔の発育にも影響する可能性があります。**

---

## 結論（先に）

- 口呼吸と不正咬合には**統計的な関連がある**（複数のシステマティックレビュー）
- **開咬、狭窄歯列、交叉咬合、ロングフェイス**との関連が報告
- ただし、**因果関係の証明は不十分**（観察研究が中心）
- 原因はアデノイド肥大、アレルギー性鼻炎など多岐にわたる
- **原因の治療（耳鼻科）+ 歯列の管理（歯科・矯正科）**の連携が重要

---

## 口呼吸とは

### 定義

正常な呼吸は**鼻呼吸**です。口が閉じていて、舌が上顎に軽く接触している状態。口呼吸とは、日常的に口を開けて呼吸する状態を指します。

### 原因

| 原因 | 説明 |
|:---|:---|
| **アデノイド肥大** | 咽頭扁桃の肥大により鼻腔からの気道が狭窄 |
| **アレルギー性鼻炎** | 鼻粘膜の腫脹により鼻呼吸が困難 |
| **鼻中隔湾曲** | 鼻腔の構造的狭窄 |
| **慢性副鼻腔炎** | 鼻閉の原因 |
| **習慣性口呼吸** | 上記の原因が解消された後も口呼吸の習慣が残る |

## エビデンス：口呼吸と不正咬合の関連

### Grippaudo et al.（2016）システマティックレビュー

- 口呼吸と以下の不正咬合との有意な関連を報告：
  - **上顎歯列の狭窄**（上あごが狭い）
  - **交叉咬合**
  - **開咬**
  - **Class II不正咬合**
  - **ロングフェイス（長顔型）**

### Zhao et al.（2021）メタアナリシス

- 口呼吸者の不正咬合リスクは、鼻呼吸者と比較して**OR 2.5〜4.0程度**上昇

### 提唱されるメカニズム

1. **舌位の変化**: 口呼吸では舌が下顎に落ち、上顎に対する舌圧が失われる → 上顎の横方向の成長が妨げられ、狭窄歯列に
2. **頬の圧力バランス**: 口が開いた状態では、頬の内向きの圧力が勝ち、上顎が狭くなる
3. **下顎の後下方回転**: 口を開けた状態での下顎位が長顔型の顔面発育につながる
4. **筋肉の不均衡**: 口唇閉鎖力の低下、頤筋の過緊張

## 口呼吸の兆候 — 保護者がチェックできること

- [ ] 普段、口が開いている
- [ ] 睡眠中にいびきをかく
- [ ] 口臭がある
- [ ] 唇が乾燥しやすい
- [ ] 朝起きたとき口が渇いている
- [ ] 食事中にくちゃくちゃ音がする
- [ ] 鼻づまりが多い
- [ ] 目の下にクマがある（アデノイド顔貌）

---

## 対応 — 多職種連携が必要

### 1. 原因の特定と治療（耳鼻科）

- アデノイド肥大 → 経過観察または手術（アデノイド切除術）
- アレルギー性鼻炎 → 薬物療法（抗ヒスタミン薬、ステロイド点鼻薬）
- 鼻中隔湾曲 → 必要に応じて手術

### 2. 歯列の管理（矯正歯科）

- 上顎の狭窄が認められる場合 → **急速拡大装置**で上顎を広げる
- これにより鼻腔底も拡大され、鼻腔容積が増加する（呼吸改善の報告もある）

### 3. 口腔筋機能療法（MFT）

- 舌位の改善、口唇閉鎖力の強化
- 口呼吸から鼻呼吸への移行をサポート
- エビデンスは限定的だが、害がなく合理的なアプローチ

---

## エビデンスの限界 — 正直にお伝えします

### 因果関係は証明されていない

- 観察研究がほとんどであり、「口呼吸が不正咬合を引き起こす」という因果関係は厳密には未証明
- 逆の因果関係（不正咬合→口唇閉鎖困難→口呼吸）の可能性もある
- 遺伝的な顎顔面の形態が共通因子として両方に影響している可能性

### 介入研究の不足

- 「口呼吸を改善したら不正咬合が予防された」というRCTはほとんどない
- アデノイド切除後の顔面成長の改善を示す研究はあるが、小規模

---

## 当院からのご提案

- **口呼吸のサインに気づいたら早めにご相談ください** — 歯並びへの影響を評価します
- **耳鼻科との連携** — 必要に応じて耳鼻科受診をお勧めします
- **上顎が狭い場合は拡大治療の検討** — 成長期を利用した介入が有効です
- **口呼吸の改善は歯だけでなく全身の健康にも** — 睡眠の質、集中力にも影響します
