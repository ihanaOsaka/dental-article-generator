# 「妊娠中にカルシウムをたくさん摂れば、赤ちゃんの歯が丈夫になる」って本当？

「赤ちゃんの歯を丈夫にするために、妊娠中は牛乳をたくさん飲まなきゃ！」――妊娠中のお母さんなら、一度はこんな話を耳にしたことがあるのではないでしょうか。

おなかの赤ちゃんのためにできることは何でもしたい、というのは自然な気持ちです。でも、この「カルシウムをたくさん摂れば歯が丈夫になる」という思い込みには、実は少し注意が必要です。

この記事では、**妊娠中のカルシウム摂取と赤ちゃんの歯の関係**について、科学的にわかっていることを正直にお伝えします。「結局どうすればいいの？」という疑問にも、具体的なアドバイスとともにお答えしていきます。

---

## おなかの中で、もう歯は作られているの？

はい、実はそうなのです。赤ちゃんの乳歯は、お母さんが「妊娠したかも？」と気づき始める頃にはもう作り始められています。

具体的には、**妊娠7〜8週頃**から歯胚（しはい）——いわば歯の「芽」にあたるもの——が形成されます。この小さな芽がやがて歯の形になり、妊娠中期から後期にかけて**カルシウムやリンといったミネラルが沈着**して、少しずつ硬い組織（歯の本体）へと成長していきます。

こうした仕組みを知ると、「じゃあ、お母さんの食事でカルシウムをしっかり摂れば、赤ちゃんの歯がもっと丈夫になるんじゃないの？」と考えるのはとても自然なことです。

しかし、この問いに対する科学的な答えは、多くの方が期待するものとは**少し違う**のです。

---

## カルシウムの推奨量は、妊娠しても増えないの？

意外に思われるかもしれませんが、**妊娠中だからといってカルシウムをいつもより多く摂る必要はない**、というのが現在の公的な見解です。

[厚生労働省「日本人の食事摂取基準（2020年版）」](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/eiyou/syokuji_kijyun.html)によると、妊婦のカルシウム推奨量は**成人女性と同じ650mg/日**。妊娠したからといって「付加量」——通常よりも多く摂ってくださいという追加の推奨——は設定されていません。

### どうして増やさなくていいの？

その理由は、私たちの体が持つ**優れた適応能力**にあります。

妊娠中は、**腸からのカルシウム吸収率が自然に上昇する**ことがわかっています。つまり、同じ量のカルシウムを食事から摂っていても、体がより効率的にカルシウムを取り込めるようになるのです。赤ちゃんの発育のために多くのカルシウムが必要になる分を、体が自動的に調整してくれている——そう考えるとわかりやすいかもしれません。

ですから、妊娠したからといって特別にカルシウムを大量に摂る必要はなく、**通常のバランスの良い食事で十分な量を確保できる**というわけです。

---

## 「カルシウムをたくさん摂れば、赤ちゃんの歯がもっと丈夫になる」って本当？

ここで、正直にお伝えしなければならないことがあります。

**「妊娠中のカルシウム摂取量を増やすことで、生まれてくる赤ちゃんの歯質が向上する」あるいは「むし歯のリスクが低下する」ということを直接的に示した、質の高いエビデンス（科学的根拠）は、現時点では十分にありません。**

### これはどういう意味？

胎児の歯の発育にカルシウムが必要であること自体は、生理学的に——つまり体の仕組みとして——明らかです。しかし、ポイントは**「通常の食事から十分なカルシウムが摂れていれば」**という条件がつくことです。

普段の食事で推奨量（650mg/日）程度のカルシウムを摂れている方が、さらにそれ以上の量を摂ったとしても、赤ちゃんの歯の質が「もっと良くなる」という科学的な裏づけは、今のところ示されていないのです。

### カルシウムが足りないとどうなるの？

では反対に、カルシウムが**極端に不足した場合**はどうなるのでしょうか。

私たちの体には、赤ちゃんの発育を最優先にする仕組みが備わっています。カルシウムが足りなくなると、体はお母さん自身の**骨からカルシウムを取り出して**、胎盤を通じて赤ちゃんに届けます。これは赤ちゃんを守るための体の防御反応ですが、お母さんにとっては**骨密度の低下**——つまり骨がもろくなること——につながる可能性があります。

さらに重度のカルシウム不足が続くと、理論的には胎児の歯の発育にも影響を与える可能性が考えられます。ただし、通常の食生活を送っている日本の妊婦さんで、そこまでの深刻な欠乏が起こることは稀です。

### 気をつけたいのは「足りていない人が多い」という事実

ここで知っておいていただきたい大切なことがあります。日本人のカルシウム摂取量は、**すべての年代で推奨量を下回る傾向にある**のです。つまり、妊娠する前からすでにカルシウムが不足している方も少なくありません。

ですから、「カルシウムを増やして歯を丈夫にしよう」と意気込むよりも、まずは**「不足しないようにする」**という意識を持つことのほうがずっと大切です。

## 赤ちゃんの歯のために、家庭でできることは？

### ステップ1：毎日の食事でカルシウムを意識する

1日650mgのカルシウムというと、ちょっとピンとこないかもしれません。以下の食品の目安を参考にしてみてください。

| 食品グループ | 食品と量の目安 | カルシウム量 |
|:---|:---|:---|
| **乳製品** | 牛乳コップ1杯（約200ml） | 約220mg |
| | ヨーグルト1個 | 約120mg |
| **小魚** | しらす大さじ2杯 | 約50mg |
| | ししゃも2尾 | 約130mg |
| **大豆製品** | 木綿豆腐1/2丁 | 約180mg |
| | 納豆1パック | 約45mg |
| **緑黄色野菜** | 小松菜1/3束 | 約130mg |
| | チンゲン菜1株 | 約100mg |

**たとえば**、朝に牛乳1杯（220mg）、昼に豆腐の味噌汁（180mg）、夕食に小松菜のおひたし（130mg）、間食にヨーグルト（120mg）を食べれば、合計650mgに届きます。

毎食1品ずつカルシウムを含む食品を取り入れることを意識すれば、無理なく推奨量に近づけるのです。

### ステップ2：カルシウム「だけ」に偏らない

赤ちゃんの歯の発育に関わる栄養素は、カルシウムだけではありません。

- **ビタミンD** — カルシウムが腸から吸収されるのを助ける、いわば「カルシウムの案内役」のような栄養素です。日光浴や魚類から摂ることができます
- **リン** — カルシウムと一緒に歯や骨の主成分になるミネラルです。肉、魚、乳製品に多く含まれます
- **タンパク質** — 歯の土台を作るために欠かせません
- **ビタミンA** — エナメル質（歯の一番外側の硬い層）の形成に関わります。にんじんやほうれん草などの緑黄色野菜に豊富です

特定の栄養素だけを大量に摂るよりも、**バランスの良い食事全体を心がけること**が、お母さんにとっても赤ちゃんにとっても最も大切です。

### ステップ3：サプリメントは「補助」として上手に活用する

つわりがひどくて食事が思うように摂れない時期や、どうしても食事だけで十分なカルシウムを摂ることが難しい場合は、サプリメントの活用も選択肢のひとつです。

ただし、**ここで注意していただきたいことがあります。** カルシウムは「多ければ多いほど良い」というものではありません。

- カルシウムの**耐容上限量**——「これ以上は摂りすぎです」という上限——は**2,500mg/日**です
- 過剰に摂ると、**便秘や腎結石**（腎臓に石ができること）のリスクが高まる可能性があります
- 自己判断で大量に摂ることは避け、**かかりつけの産婦人科医に相談**してから利用しましょう

---

## 結局、何が一番大切なの？ — まとめ

ここまでの内容を整理しましょう。

- 妊娠中のカルシウム推奨量は**650mg/日**で、妊娠していない時と同じです
- 妊娠中は体の仕組みとしてカルシウムの吸収率が自然に上がるため、**追加で増やす必要はありません**
- 「カルシウムをたくさん摂れば赤ちゃんの歯がより丈夫になる」という**直接的なエビデンスは、現時点では十分ではありません**
- 大切なのは、カルシウムを**不足させないこと**と、**バランスの良い食事を心がけること**
- そして忘れてはならないのは、赤ちゃんのむし歯予防にとって最も重要なのは、**生まれた後のケア**——食習慣の管理、毎日の歯みがき、フッ素（フッ化物）の適切な活用——であるということです

妊娠中の食事は赤ちゃんの全身の健康に関わる大切なテーマです。過度に心配したり、「あれも摂らなきゃ、これも摂らなきゃ」とプレッシャーに感じたりする必要はありません。

むしろ、**日頃の食事を見直すよい機会**として、楽しみながら取り組んでいただけたらと思います。
